1.2 Generative¶

Automatic Chemical Design Using a Data-Driven Continuous Representation of Molecules
ACS central science 4, no. 2 (2018): 268-276. Gómez-Bombarelli, Rafael, Jennifer N. Wei, David Duvenaud, José Miguel Hernández-Lobato, Benjamín Sánchez-Lengeling, Dennis Sheberla, Jorge Aguilera-Iparraguirre, Timothy D. Hirzel, Ryan P. Adams, and Alán Aspuru-Guzik.
Sketch: AutoEncoder: SMILES string–>latent vector–>SMILES string. Convert discrete molecules to a multidimensional continuous representation. Generate new molecules for efficient exploration and optimization through open-ended spaces of chemical compounds.
Generating Focused Molecule Libraries for Drug Discovery with Recurrent Neural Networks
ACS central science 4, no. 1 (2018): 120-131. Segler, Marwin HS, Thierry Kogej, Christian Tyrchan, and Mark P. Waller.
Sketch: Stacked LSTMs, SMILE–>LSTM–>SMILE. Generate molecule SMILEs via stacked LSTMs. Fine tune this model on smaller dataset to get desired molecules, so called transfer learning.
Multi-Objective Molecule Generation using Interpretable Substructures
ICML(2020) Jin, Wengong, Regina Barzilay, and Tommi Jaakkola.
Sketch: Mix molecules with different desired properties to produce the final compound.
Code: Github(PyTorch) Errors. The code doesn’t works, possiblely because the different version of the ‘rdkit’ package.
Improving Molecular Design by Stochastic Iterative Target Augmentation
ICML(2020) Somnath, Vignesh Ram, Charlotte Bunne, Connor W. Coley, Andreas Krause, and Regina Barzilay.`
Sketch: Set2Set and HierGNN
Code: Github(PyTorch) Errors. Due to the unstable updation of chemprop, the code doesn’t work.
Hierarchical Generation of Molecular Graphs using Structural Motifs
ICML(2020) Jin, Wengong, Regina Barzilay, and Tommi Jaakkola.
Sketch: Hierarchical VAE
Code: Github(PyTorch) Errors. The code doesn’t works, possiblely because the different version of the ‘rdkit’ package.
A Generative Model for Molecular Distance Geometry
ICML(2020) Simm, Gregor NC, and José Miguel Hernández-Lobato.
Sketch: Generate 3d structure from graph via CVAE
Code: Github(Tensorflow)
Graphaf: a flow-based autoregressive model for molecular graph generation
ICLR(2020) Shi, Chence, Minkai Xu, Zhaocheng Zhu, Weinan Zhang, Ming Zhang, and Jian Tang.
Sketch: TODO
Code: Github(PyTorch) Good!

1.3 Predictive¶

Convolutional Networks on Graphs for Learning Molecular Fingerprints
NIPS 2015 Duvenaud, David, Dougal Maclaurin, Jorge Aguilera-Iparraguirre, Rafael Gómez-Bombarelli, Timothy Hirzel, Alán Aspuru-Guzik, and Ryan P. Adams.
Sketch: Learning the graph embedding with message passing.
Code: Github(PyTorch) Github(Theano+Keras)
Molecule Attention Transformer
NeurIPS(workshop) 2019 Maziarka, Łukasz, Tomasz Danel, Sławomir Mucha, Krzysztof Rataj, Jacek Tabor, and Stanisław Jastrzębski.
Sketch: Argument the attention matrix with distance and ajacency matrix.
Code: Github(Pytorch) Good!
Directional Message Passing for Molecular Graphs
ICLR(2020) Klicpera, Johannes, Janek Groß, and Stephan Günnemann.
Sketch: TODO
Code: Github(Tensorflow)
ProteinGCN: Protein model quality assessment using graph convolutional networks
BioRxiv 2020 Sanyal, Soumya, Ivan Anishchenko, Anirudh Dagar, David Baker, and Partha Talukdar.
Sketch: GCN+Pooling
Code: Github(Pytorch)
Heterogeneous Molecular Graph Neural Networks for Predicting Molecule Properties
ICDM 2020 Shui, Zeren, and George Karypis.
Sketch: Heterogeneous Molecular Graph Neural Networks. High-order graph convolution, considering interactions between functional groups
Code: Github(PyTorch)
TrimNet: learning molecular representation from triplet messages for biomedicine
Briefings in Bioinformatics (2020) Li, Pengyong, Yuquan Li, Chang-Yu Hsieh, Shengyu Zhang, Xianggen Liu, Huanxiang Liu, Sen Song, and Xiaojun Yao.
Sketch: A variant of GAT(Graph Attention model)+Set2Set+Focal loss. Use graph attention machanism to learn node features, then aggregate them with Set2Set, and finally optimize the model via Focal loss.
Code: Github(PyTorch)
Molecular Mechanics-Driven Graph Neural Network with Multiplex Graph for Molecular Structures
NIPS 2020 Zhang, Shuo, Yang Liu, and Lei Xie.
Sketch: Hierarchical graph convolution, i.e., local and global message passing.
Code: Github(PyTorch) Good!

1.4 Self-supervised¶

Infograph: Unsupervised and semi-supervised graph-level representation learning via mutual information maximization
ICLR(2020) Sun, Fan-Yun, Jordan Hoffmann, Vikas Verma, and Jian Tang.
Sketch: Adversarial self-supervised loss + supervised loss. aximize Mutual Information at each layer.
Code: Github(PyTorch)
Self-Supervised Graph Transformer on Large-Scale Molecular Data
NIPS(2020) Rong, Yu, Yatao Bian, Tingyang Xu, Weiyang Xie, Ying Wei, Wenbing Huang, and Junzhou Huang.
Sketch: Transformer + GNN + node/edge/graph level self-supervised tasks. Downstream tasks include classification and regression.
Code: Github(PyTorch)

1.5 Retrosynthesis¶

Learning Graph Models for Template-Free Retrosynthesis
ICML workshop(2020) Somnath, Vignesh Ram, Charlotte Bunne, Connor W. Coley, Andreas Krause, and Regina Barzilay.
Sketch: Use two stage methods to achieve better results: (1) predict edit (2) add leaving groups
Code: Github(PyTorch) Minor errors. The code works with a few tweaks.
